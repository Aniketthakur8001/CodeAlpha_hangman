# CodeAlpha_hangman

A simple text-based Hangman game built in Python.  
This project is part of my CodeAlpha internship tasks.

## 📌 Features
- Random word selection
- Limited incorrect attempts (6)
- Tracks used letters
- Fun console-based gameplay
